How it works

Track. Compare. Buy smarter.

01
🧩

Install the extension

Add Index748 to Chrome in one click. It runs quietly in the background while you browse vehicle listings as normal. Firefox support coming soon.

02
📊

We build the picture

Every listing you view is anonymously captured — make, model, year, price, mileage. Over time this becomes a real-time SA vehicle price index.

03
🎯

See the full market

Search any make, model, and year range. See growing price history, demand signals, deal scoring, and instant alerts when prices drop. The longer Index748 runs, the more valuable it gets.
